`静态方法`，一般用于写工具。
```ts
class Person {
    static say() {
        console.log("success")
    }
}

Person.say()
```

`多态`，父类定义一个方法不去实现，由继承他的子类去实现方法，每个子类有不同的表现。

```ts
class Animal {
    eat() {
        console.log('实现吃的方法(子类也可以不重现)')
    }
}

class Dog extends Animal {
    name: string
    constructor(name: string) {
        super()
        this.name = name
    }
    eat() {
        console.log(`${this.name}吃肉`)
    }
}

class Cat extends Animal {
    name: string
    constructor(name: string) {
        super()
        this.name = name
    }
    eat() {
        console.log(`${this.name}吃鱼`)
    }
}

const cat = new Cat("猫")
cat.eat()
const dog = new Dog("狗")
dog.eat()
```

抽象类和抽象方法。抽象方法必须要在抽象类内，抽象类内可以没有抽象方法。
```ts
abstract class Animal {
    // 子类必须要实现eat方法
    abstract eat(): any;
}

class Dog extends Animal {
    name: string
    constructor(name: string) {
        super()
        this.name = name
    }
    eat() {
        console.log(`${this.name}吃肉`)
    }
}

class Cat extends Animal {
    name: string
    constructor(name: string) {
        super()
        this.name = name
    }
    eat() {
        console.log(`${this.name}吃鱼`)
    }
}

const cat = new Cat("猫")
cat.eat()
const dog = new Dog("狗")
dog.eat()
```
